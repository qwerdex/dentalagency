Invisalign Full

Основной продукт Invisalign, полный метод лечения для всех аномалий окклюзии.

Неограниченное количество капп и возможность использовать всё, что умеет Invisalign (любые аттачменты и детали элайнеров), для выравнивания ваших зубов.

Invisalign Teen

Полный метод лечения, разработанный специально для подростков.

Благодаря этой системе начинать лечение можно даже при наличии молочных зубов и при не полностью прорезавшихся постоянных. Для контроля за ношением элайнеров на каппе есть индикаторы ношения — синие точки, которые светлеют по мере ношения элайнера. С такими индикаторами вы можете не переживать, что ребенок снимет каппы в школе.

Invisalign Lite

Продукт, разработанный для исправления легких клинических случаев.

Имеются ограничения по количеству капп — не более 14 этапов лечения. Однако не всеми особенностями системы и деталями элайнера можно воспользоваться. Продукт подойдет для коррекции небольших несовершенств. Целесообразность его использования вы определите при встрече с лечащим врачом.

Invisalign Vivera retainers

Ретейнеры для удержания полученного результата лечения.

После окончания любого ортодонтического лечения необходимо пройти период закрепления результата, который называется ретенционным. Vivera — ретенционные каппы, которые надеваются только на ночь для закрепления полученного результата.
